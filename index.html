利用hexo+github搭建静态博客

Posted on 2019-09-28

从零开始搭建博客

安装git

安装node.js

下载node.js

安装完成后会附带npm包管理器

为了防止后续下载速度过慢，使用淘宝镜像

安装hexo

生成博客

先创建一个blog文件夹

写一篇博文

编写完一篇博文后

安装部署插件

修改配置文件

修改blog目录下的_config.yml文件

将_config.yml文件最底部修改为：

deploy:
    type: git
    repo: https://github.com/Sakura-cheng/Sakura-cheng.github.io.git
    branch: master

部署到远端

修改主题

修改_config.yml文件中

theme: next
